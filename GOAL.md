# The goal of this program

This program should achieve the following:
 - Help a user find a parking spot in a given parking garage
   - This spot must be chosen in the most optimal way out of a set of possible choices
 - Help the user navigate directly from their position to the parking spot
 - Allow parking garage owners to use this product to help manage their parking garage
   - The program must then be able to represent a number of layouts, encoded in a file
